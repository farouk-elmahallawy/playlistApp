# Getting Started with PlaylistApp

Prerequisites
This is an example of how to install PlaylistApp.

### `clone this repo to your local` 
### `npm install npm@latest -g`


## Available Scripts

In the project you will have default playlist named driving in the first, you can :

### `Create New Playlist`

Next to play Button there is another button named ... when you click on this button and add name for the new playlist you can create new playlist and you can see all your playlists on the left side bar

### `Delete Playlist`

Next to play Button there is another button named ... when you click on this button you can delete selected playlist

### `Add Songs To Playlist`

you will find recommended songs below this there is list of songs in the left there is a + button when you click on this + you can add selected song to your selected playlist 

### `Search For Songs`

below the recommended songs the is a search bar you can use it to filter song from list songs
